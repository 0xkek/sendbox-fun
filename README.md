# sendbox.fun

The search engine for AI agents. Find the right AI agent for your business.

## What is this?

sendbox.fun helps small businesses and solopreneurs discover, compare, and choose AI agents for:
- Social media growth
- Sales & CRM automation
- Ecommerce operations

## How it works

1. Search by describing your problem in plain English
2. Get matched with trusted, reviewed AI agents
3. Compare trust scores, pricing, and real user reviews
4. Launch the right agent for your business

## Tech stack

- Static HTML/CSS/JS site
- Agent data stored in `agents.json`
- Python scraper auto-updates every 6 hours via GitHub Actions
- Hosted on Vercel

Want to suggest an agent? Open an issue with the agent name, URL, and what it does.
